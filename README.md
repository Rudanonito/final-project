# US Housing Market Analysis - County Level

## Project Overview
This project analyzes the relationship between housing prices, unemployment rates, income levels, poverty rates, and educational attainment across US counties using data from Zillow, BLS, and US Census Bureau.
## Project Structure
- `src/main.py` - Main pipeline execution script

- `src/config.py` - Configuration constants and URLs

- `src/data_loading.py` - Functions for loading data from APIs

- `src/data_cleaning.py` - Functions for cleaning and preparing data

- `src/data_merging.py` - Functions for merging datasets

- `src/analysis.py` - Statistical analysis and visualization functions

- `src/tests.py` - Unit tests for data processing functions



## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run tests: `python tests.py`
3. Run main: `python main.py`
