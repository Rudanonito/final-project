# US Housing Market Analysis - County Level

Analysis of socio-economic factors on US housing markets at county level using Zillow, Census, and BLS data.

## Project Structure
- `main.py` - Main data loading and processing scripts
- `test.py` - Test scripts for data validation
- `config.py` - Configuration file with API keys (You should rename `config_example.py` and put your API)
- `requirements.txt` - Python dependencies

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run tests: `python test.py`
3. Run main: `python main.py`

- Zillow Home Value Index
- US Census Bureau API
